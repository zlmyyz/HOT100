# 颜色分类

数组中有三类数0，1，2；按照012顺序排列

```
class Solution {
    public int[] sortColors(int[] nums) {
        int n=nums.length;
        int a0=0,a1=0,a2=0;
        for(int i=0;i<n;i++){
            if(nums[i]==0)a0++;
            else if(nums[i]==1)a1++;
            else a2++;
        }
        for(int i=0;i<n;i++){
            if(a0!=0){nums[i]=0;a0--;}
            else if(a1!=0){nums[i]=1;a1--;}
            else {nums[i]=2;a2++;}
        }
        return nums;
    }
}
```
